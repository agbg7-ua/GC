# 🏙️ Ciudad Japonesa para OpenGL (Proyecto Interactivo)

Este proyecto consiste en el modelado de una ciudad de estilo japonés desarrollada en Blender, pensada para su implementación en OpenGL como entorno interactivo en tiempo real.

---

## 📌 Descripción

El escenario representa una ciudad de estilo japonés que combina elementos tradicionales y modernos dentro de una estructura urbana organizada por zonas. La escena está diseñada para ser utilizada como entorno interactivo en un sistema de renderizado en OpenGL.

---

## 🗺️ Distribución del escenario

La ciudad está organizada en cuatro zonas principales, cada una ubicada en una esquina del mapa:

- 🛕 **Esquina 1: Pagoda**
  Punto tradicional y emblemático del escenario, funcionando como referencia visual principal.

- 🌳 **Esquina 2: Parque japonés**
  Zona natural con vegetación, senderos y estética inspirada en jardines japoneses.

- 🏢 **Esquina 3: Edificios residenciales**
  Área urbana con bloques de viviendas y arquitectura moderna.

- 🏪 **Esquina 4: Tiendas y zona comercial**
  Calle comercial con tiendas, carteles y ambiente urbano activo.

---

## 🚗 Funcionalidad prevista

El proyecto está diseñado para un entorno interactivo donde:

- Un coche puede moverse libremente por la ciudad
- La cámara sigue el movimiento del coche en tercera persona
- El usuario puede explorar las cuatro zonas del mapa
- El entorno funciona como escenario en tiempo real dentro de OpenGL

---

## 🧱 Contenido del proyecto

- Modelado completo de ciudad japonesa dividida por zonas
- Pagoda como punto tradicional principal
- Parque con diseño de jardín japonés
- Zona residencial con edificios urbanos
- Zona comercial con tiendas y cartelería
- Estructura optimizada para gráficos en tiempo real

---

## 🛠️ Tecnologías

- Blender (modelado 3D)
- OpenGL (renderizado en tiempo real)

---

## 🎯 Objetivo

Este proyecto tiene como finalidad servir como base para un entorno interactivo en OpenGL, aplicando conceptos de:

- Gráficos por computadora en tiempo real
- Organización de escenas 3D por zonas
- Movimiento de vehículo en entorno urbano
- Cámara dinámica tipo videojuego
- Optimización de assets para rendimiento

---

## ⚠️ Notas

- El modelo está optimizado para uso en tiempo real.
- Se recomienda mantener baja la complejidad geométrica en la exportación.
- Las texturas deben estar comprimidas para mejorar rendimiento.

# 🏙️ Japanese City for OpenGL (Interactive Project)

This project consists of a Japanese-style city modeled in Blender, designed specifically for integration into OpenGL as a real-time interactive environment.

---

## 📌 Description

The scene represents a Japanese-inspired city that blends traditional and modern elements within a structured urban layout. It is designed to function as an interactive environment rendered in real time using OpenGL.

---

## 🗺️ Scene Layout

The city is organized into four main areas, each located in a different corner of the map:

- 🛕 **Corner 1: Pagoda**
  A traditional landmark serving as the main visual reference point of the scene.

- 🌳 **Corner 2: Japanese Park**
  A natural area featuring vegetation, walking paths, and a Japanese garden-inspired aesthetic.

- 🏢 **Corner 3: Residential Buildings**
  An urban district with apartment blocks and modern residential architecture.

- 🏪 **Corner 4: Shops and Commercial Area**
  A lively street filled with stores, signage, and a typical Japanese commercial atmosphere.

---

## 🚗 Intended Functionality

The project is designed for an interactive environment where:

- A car can freely move through the city
- The camera follows the car in third-person view
- The user can explore all four zones of the map
- The environment behaves like a real-time OpenGL scene

---

## 🧱 Project Content

- Full Japanese-style city layout divided into zones
- Pagoda as the main traditional landmark
- Japanese garden-inspired park area
- Residential urban district with buildings
- Commercial shopping street with signage
- Optimized structure for real-time rendering

---

## 🛠️ Technologies

- Blender (3D modeling)
- OpenGL (real-time rendering)

---

## 🎯 Objective

The goal of this project is to serve as a foundation for an interactive OpenGL environment, applying concepts such as:

- Real-time computer graphics
- Zone-based 3D scene organization
- Vehicle movement in an urban environment
- Third-person camera systems
- Asset optimization for performance

---

## ⚠️ Notes

- The model is optimized for real-time usage.
- Low-polygon geometry is recommended for export.
- Textures should be compressed for better performance.
